# Health & Fitness Tracking Chatbot

## Overview
This project is a Python-based chatbot designed to help users track their fitness progress, including weight, workouts, diet, and water intake. It was created with athletes in mind, especially those managing training and weight goals.

## Features
- Log and view weight progress
- Track workouts and water intake
- Get workout and diet suggestions
- Add custom exercises and diet options
- Persistent data storage using CSV files

## Technologies Used
- Python
- pandas
- CSV file handling
- Functions and modular programming
- Loops and conditional statements
- Input validation and error handling

## Purpose
This project was inspired by my experience as a wrestler, where tracking fitness and weight changes is essential. It solves a real-world problem of staying organized and consistent with health goals.

## How It Works
The program runs through a menu-based system where users select options to log or view data. All information is saved to CSV files, so progress is not lost after closing the program.
